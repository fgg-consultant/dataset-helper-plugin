How to use this page
This page manages the WMS layers available for your Climweb map viewer. The tree below shows all layers grouped by Category and Subcategory.
  • Use the checkboxes to enable or disable individual layers.
  • Status indicators: synced (already in Climweb), will be added on next load, will be removed on next load, inactive.
  • Click Synchronize with Climweb to apply your selection — enabled layers are created in the database, disabled ones are removed.
  • You can extend the catalog by adding layers manually, importing from a remote WMS service, or loading a JSON configuration.

Bulk Import

Import multiple categories, subcategories, and datasets from a JSON configuration file.

View sample JSON format 
{
  "categories": [
    {
      "title": "Vegetation",
      "icon": "leaf",
      "subcategories": [
        {
          "title": "NDVI Products",
          "datasets": [
            {
              "title": "NDVI from CGLS (1km)",
              "description": "Normalized Difference Vegetation Index",
              "multi_temporal": true,
              "metadata": {
                "title": "NDVI from CGLS",
                "function": "Vegetation greenness indicator",
                "resolution": "1km",
                "geographic_coverage": "Africa",
                "source": "Copernicus Global Land Service",
                "frequency_of_update": "Dekadal"
              },
              "layers": [
                {
                  "type": "wms",
                  "title": "NDVI Default",
                  "layer_name": "vgt-ndvi_vgt-pv-olci_ndvi",
                  "wms_url": "https://estation.jrc.ec.europa.eu/geoserver/wms",
                  "default": true
                }
              ]
            }
          ]
        }
      ]
    }
  ]
}
